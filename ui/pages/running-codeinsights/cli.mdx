# Usage from command line

```bash
$ npx @codeinsights/js --repoPath "/path/to/repo"
```

## Command Line Arguments

| Name               | Default Value | Description                                            |
|---------------|---------------|--------------------------------------------------------|
| repoPath    | `null`          | Absolute path of the repository or GitHub URL          |
| configName  | `analyzer.config.(js\|ts\|json)`            | Name of the configuration file, if exists          |
